## 🚀 快速开始

### 1. 启动服务器

**Windows用户**:
```bash
# 双击运行
start-server.bat
```

**Mac/Linux用户**:
```bash
python3 -m http.server 8000
```

### 2. 访问网站
```
http://localhost:8000
```
